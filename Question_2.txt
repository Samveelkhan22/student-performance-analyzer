VARIABLE:
Definition: A named storage location in a program that holds a value.
Example: first_test, second_test, and third_test are variables that store test scores.

FUNCTION DEFINITION:
Definition: A block of code that performs a specific task, defined using the def keyword.
Example:
def calc_average(score1, score2, score3):
    return (score1 + score2 + score3) / 3

PARAMETER:
Definition: A variable used in a function to receive and store values passed during a function call.
Example:

    # score1, score2, and score3 are parameters in this function

STRING:
Definition: A sequence of characters enclosed in single or double quotes.
Example: "Enter 3 test scores:", "You scored all As", "Your letter grade is A"

ASSIGNMENT:

Definition: The process of giving a value to a variable using the = operator.
Example:
first_test = get_grade()

ORDER OF OPERATIONS:

Definition: A set of rules specifying the sequence in which operations are performed in a mathematical expression.
Example:
return (score1 + score2 + score3) / 3
